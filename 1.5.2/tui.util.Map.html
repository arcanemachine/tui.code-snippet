    tui.util.Map(initData)

    new Map(initData)
    map.js, line 98

    The Map object implements the ES6 Map specification as closely as possible.
    For using objects and primitive values as keys, this object uses array internally.
    So if the key is not a string, get(), set(), has(), delete() will operates in O(n),
    and it can cause performance issues with a large dataset.

    Features listed below are not supported. (can't be implented without native support)

    • Map object is iterable
    • Iterable object can be used as an argument of constructor

    If the browser supports full implementation of ES6 Map specification, native Map obejct will be used internally.

    Parameters:
    Name Type Description
    initData Array

    Array of key-value pairs (2-element Arrays). Each key-value pair will be added to the new Map
    Examples
    // node, commonjs
var Map = require('tui-code-snippet').Map;
    // distribution file, script
