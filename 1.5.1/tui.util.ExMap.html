    new ExMap(initData)
    exMap.js, line 34

    The ExMap object is Extended Version of the tui.util.Map object.
    and added some useful feature to make it easy to manage the Map object.

    Parameters:
    Name Type Description
    initData Array

    Array of key-value pairs (2-element Arrays). Each key-value pair will be added to the new Map
    Examples
    // node, commonjs
var ExMap = require('tui-code-snippet').ExMap;
    // distribution file, script
